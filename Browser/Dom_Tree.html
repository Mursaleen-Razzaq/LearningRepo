DOM LEARNING

Why the DOM Tree is Important.
1. It lets JavaScript read and modify HTML.
2. It enables dynamic web pages without reloading.
3. It allows handling user interactions (clicks, forms, keyboard events).
4. It supports creating, deleting, and updating elements at runtime.
In short, the DOM Tree is the hierarchical structure of an HTML document,
and JavaScript uses it to interact with and manipulate web page content.
